개인정보처리방침

시행일: 2026년 5월 29일 · 마지막 업데이트: 2026년 5월 29일

본 방침은 Custom Animation Alert (이하 "앱")이 사용자의 개인정보를 어떻게 처리하는지 설명합니다. 앱을 설치·사용함으로써 본 방침에 동의한 것으로 간주됩니다.

1. 한눈에 보기 (요약)

  • 알림 내용은 사용자 기기에서만 처리되며 외부로 전송되지 않습니다.
  • 마켓 기능(애니메이션 공유)을 사용할 때만 Google 계정으로 로그인이 필요하며, 공유한 콘텐츠와 닉네임이 Firebase에 저장됩니다.
  • 광고나 분석 SDK를 사용하지 않으며 사용자 행동을 추적하지 않습니다.
  • 언제든 계정 삭제로 모든 데이터를 영구 삭제할 수 있습니다.

2. 수집·처리하는 정보

2.1 기기 내에서만 처리되고 외부 전송되지 않는 정보

정보목적저장 위치
수신 알림의 패키지명·제목·본문사용자가 정의한 규칙과 매칭기기 메모리 (저장되지 않음)
설치된 앱 목록규칙 생성 시 "앱 선택" UI기기 메모리 (저장되지 않음)
사용자 정의 규칙알림 매칭 설정기기 내부 저장소 (SharedPreferences)
사용자가 선택한 미디어·사운드 파일알림 발동 시 재생기기 내부 저장소

위 정보는 본 앱이 서버로 전송하지 않으며, 사용자 외 누구도 접근할 수 없습니다.

2.2 마켓 기능 사용 시 (선택)

마켓 기능(다른 사용자와 애니메이션 공유)을 이용하는 경우에만 Google 계정으로 로그인이 필요하며, 다음 정보가 Google Firebase 서버에 저장됩니다.

정보출처목적
Google 계정 UID, 이메일, 표시 이름Firebase Authentication로그인 식별, 본인 콘텐츠 관리
사용자가 직접 설정한 닉네임사용자 입력마켓에 본인 콘텐츠의 제작자명으로 표시
사용자가 공유 버튼으로 업로드한 애니메이션 규칙(JSON), 미디어 파일, 사운드 파일사용자 행동다른 사용자가 마켓에서 검색·다운로드
사용자가 "좋아요"를 누른 콘텐츠 ID 목록사용자 행동좋아요 상태 표시, 좋아요 카운트 합산
다운로드 횟수, 좋아요 수 (집계값)자동 집계마켓에서 인기 순 정렬

2.3 외부 API를 통해 조회하는 공개 정보

코인 가격 알람 기능을 사용하는 경우, 앱이 백그라운드에서 CoinGecko 공개 API에 가격을 조회합니다. 이 과정에서 사용자 식별 정보는 전송되지 않으며, 단순히 "비트코인의 현재 가격" 같은 공개 시장 데이터를 받아옵니다.

3. 제3자 제공·공유

본 앱은 사용자 개인정보를 제3자에게 판매·임대·공유하지 않습니다. 다만 다음 서비스를 통해 데이터가 처리됩니다.

  • Google Firebase (Authentication, Firestore, Storage) — 마켓 기능에 필요한 인증과 저장소 제공. Google 개인정보처리방침을 따릅니다: policies.google.com/privacy
  • CoinGecko — 코인 가격 공개 API. 식별 정보 전송 없음: coingecko.com/privacy

4. 권한 사용 사유

권한사유
알림 액세스 (BIND_NOTIFICATION_LISTENER_SERVICE)앱의 핵심 기능 — 수신 알림을 사용자 규칙과 매칭하기 위해 필수. 알림 내용은 매칭 처리에만 사용되고 외부로 전송되지 않습니다.
다른 앱 위에 표시 (SYSTEM_ALERT_WINDOW)알림 매칭 시 사용자 설정 미디어를 화면 위에 표시. 약 3~5초간만 노출되며 다른 앱 사용을 방해하지 않습니다.
앱 목록 조회 (QUERY_ALL_PACKAGES)규칙 생성 시 "앱 선택" 화면에서 설치된 앱을 보여주기 위함. 앱 목록은 표시 용도로만 사용되며 외부 전송되지 않습니다.
인터넷 (INTERNET)마켓 기능(Firebase 통신)과 코인 가격 조회(CoinGecko API)에 사용됩니다.
화면 깨우기 (WAKE_LOCK)"화면 꺼졌을 때 깨우기" 옵션 사용 시, 알림 매칭 순간 짧게 화면을 켜는 데 사용됩니다.

5. 보관 기간

  • 기기 내 저장 정보(규칙, 미디어 파일 등)는 사용자가 앱을 삭제할 때 함께 삭제됩니다.
  • 마켓 업로드 콘텐츠는 사용자가 직접 삭제하거나, 계정 삭제 요청 시까지 보관됩니다.
  • 로그 파일은 기기 내부에서만 임시 저장되며 외부 전송되지 않습니다.

6. 사용자 권리

사용자는 다음 권리를 갖습니다:

  • 열람·정정: 앱 설정 → 계정에서 본인이 업로드한 콘텐츠와 닉네임을 확인·수정할 수 있습니다.
  • 삭제: 마켓에 올린 개별 콘텐츠 삭제는 앱 내 "내가 공유한 파일"에서 가능합니다. 계정 전체 삭제는 아래 연락처로 요청해주세요.
  • 로그아웃: Google 계정 연결을 해제하면 마켓 관련 기능이 비활성화됩니다. 알림 매칭 기능은 그대로 동작합니다.

7. 아동 개인정보 보호

본 앱은 만 14세 미만 아동을 대상으로 하지 않으며, 의도적으로 만 14세 미만의 개인정보를 수집하지 않습니다. 아동의 개인정보가 수집된 사실을 인지한 경우 즉시 삭제합니다.

8. 방침 변경

본 방침이 변경되는 경우 본 페이지의 "시행일"이 갱신됩니다. 중대한 변경 시 앱 내 공지로 안내합니다.
