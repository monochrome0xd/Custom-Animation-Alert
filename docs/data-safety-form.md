# Play Console 데이터 안전(Data Safety) 양식 답변

Play Console에 입력할 답변을 정리한 문서. 양식이 자주 바뀌므로 항목 명칭이 약간 다를 수 있으나 의미는 동일.

위치: **Play Console → 앱 콘텐츠 → 데이터 보안 (Data safety)**

---

## 1. 데이터 수집 및 공유 (Data collection)

### 1.1 앱이 사용자 데이터를 수집·공유합니까?
- **예** (Yes)

---

## 2. 수집하는 데이터 항목

각 항목별로 "수집함 / 공유함 / 처리 목적 / 선택성 / 임시 처리 여부" 선택.

### 📧 Personal info → 이메일 주소 (Email address)
- 수집함: **예**
- 공유함: **아니요** (Firebase에 저장되지만 다른 사용자에게 보이지 않음)
- 처리 목적: ☑ 계정 관리 (Account management)
- 사용자가 제공: **예** (Google 로그인 시)
- 필수 데이터: **아니요** (선택적 — 마켓 기능 사용 시에만)

### 👤 Personal info → 이름 (Name)
- 수집함: **예** (Google 표시 이름)
- 공유함: **아니요**
- 처리 목적: ☑ 계정 관리
- 사용자가 제공: **예**
- 필수 데이터: **아니요**

### 🆔 Personal info → 사용자 ID (User IDs)
- 수집함: **예** (Firebase UID + 사용자가 직접 설정한 닉네임)
- 공유함: **예** (닉네임만 — 마켓에서 본인 콘텐츠 제작자로 표시)
- 처리 목적: ☑ 계정 관리, ☑ 앱 기능
- 사용자가 제공: **예**
- 필수 데이터: **아니요**

### 🎨 App activity → 앱 내 검색 기록 (App searches)
- 수집함: **아니요**

### 🖱 App activity → 사용자가 만든 콘텐츠 (User-generated content)
- 수집함: **예** (사용자가 마켓에 공유한 애니메이션 규칙, 미디어 파일, 사운드)
- 공유함: **예** (마켓 다른 사용자에게 공개)
- 처리 목적: ☑ 앱 기능
- 사용자가 제공: **예**
- 필수 데이터: **아니요** (공유 안 하면 수집 안 됨)

### 📁 Files and docs → 사진/동영상 (Photos and videos)
- 수집함: **예** (사용자가 직접 선택해서 마켓에 업로드한 경우)
- 공유함: **예** (마켓 공개)
- 처리 목적: ☑ 앱 기능
- 사용자가 제공: **예**
- 필수 데이터: **아니요**

### 🎵 Files and docs → 오디오 파일 (Audio files)
- 수집함: **예** (사용자가 마켓에 업로드한 사운드)
- 공유함: **예** (마켓 공개)
- 처리 목적: ☑ 앱 기능
- 사용자가 제공: **예**
- 필수 데이터: **아니요**

### 📱 Device or other IDs

- 수집함: **아니요**

- 수집함: **아니요**

### 📞 Contacts

### 💬 Messages

- **해당 없음** (만 14세 미만 대상 앱 아님)

### 3.4 본 앱이 독립적 보안 평가를 받았습니까?
- **아니요** (개인 개발자 앱)

## 5. 추가 메모 — 권한 사유서에서 강조할 점

위 데이터 안전 양식 외에 **민감한 권한 선언** 단계에서 다음 두 가지를 명확히 적어야 합니다.

### NotificationListener
> 알림 내용은 사용자 기기 메모리에서만 즉시 처리되며 저장·전송되지 않습니다.
> 매칭 처리 직후 폐기되어 어떤 데이터베이스에도 기록되지 않습니다.

### QUERY_ALL_PACKAGES
> 설치된 앱 목록은 사용자가 규칙 생성 시 "앱 선택" UI에서만 표시 용도로 조회되며,
> 외부 전송이나 저장이 일체 없습니다.

1. 양식 저장 후 "심사 시작" 버튼 누르기 (실제 심사는 출시 시점에 같이 진행)
2. 개인정보처리방침 URL은 다음 입력:
   ```
   https://monochrome-x.github.io/Custom-Animation-Alert/privacy-policy.html
   ```
3. 위 정책 페이지에 본 양식의 모든 항목이 일관되게 기술돼있어야 함 (이미 작성됨)
