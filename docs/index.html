Custom Animation Alert

알림을 더 눈에 띄게 만드는 안드로이드 앱.

사용자가 정의한 규칙(예: 카카오톡에서 '주문' 키워드)에 매칭되는 알림이 오면, 사전 설정한 이미지·GIF·동영상·Lottie 애니메이션과 사운드를 화면 위에 잠시 표시합니다.

주요 기능

  • 알림 키워드·앱별 매칭 규칙
  • 커스텀 미디어 (이미지, GIF, 동영상, Lottie JSON)
  • 커스텀 사운드 (파일 임포트, URL 다운로드, 컷 편집)
  • 마블 / 천천히 상승 / 방향 이동 / 살짝 보임 등 다양한 애니메이션 모드
  • 코인 가격 알람 ($BTC 같은 티커로 자동 설정)
  • 커뮤니티 마켓 (애니메이션 공유 및 다운로드)
  • 5가지 컬러 테마 + 라이트/다크 모드
